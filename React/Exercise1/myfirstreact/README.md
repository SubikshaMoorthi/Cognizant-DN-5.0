# My First React Application

## Overview
This project is a simple React application created using Create React App. It displays a welcome message to demonstrate the basic structure of a React application.

## Features
- React application setup using Create React App
- Displays a heading on the home page
- Demonstrates JSX and functional components

## Technologies Used
- React
- JavaScript (ES6)
- HTML5
- CSS3

## How to Run

```bash
npm install
npm start
