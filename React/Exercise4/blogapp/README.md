# Blog Application

## Overview
A React application demonstrating React lifecycle methods. Posts are fetched from an external REST API and displayed dynamically.

## Features
- Class component
- componentDidMount()
- componentDidCatch()
- Fetch API
- Dynamic rendering of posts

## API Used
https://jsonplaceholder.typicode.com/posts

## Technologies Used
- React
- JavaScript (ES6)
- Fetch API
- HTML5

## How to Run

```bash
npm install
npm start
