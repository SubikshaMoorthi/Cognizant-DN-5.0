# Student Management Portal

## Overview
A React application demonstrating the use of class components. The application renders multiple components representing different sections of a Student Management Portal.

- Multiple reusable components
- Component rendering using App component

## Components
- Home
- About
- Contact

- HTML5
- CSS3
